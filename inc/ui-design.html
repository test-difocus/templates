Иструкция по использованию

Для подключения стиля или смены фона необходимо нажать на пиктограмму
со схематичным изображением фона или дизайна сайта,
после появления надписи "Сохранено" - изменения внесены.

Изменение дизайна происходит при использовании визуального редактора
CSS(все разделы, кроме "CSS" и "Готовые стили"). Для сохранения изменений,
необходимо нажать кнопку сохранить.

Для редактирования CSS-кода вручную(раздел "CSS"):

  • Сохраните изменения в визуальном редакторе CSS
    и обновите страницу сайта(ctrl+F5);
  • Перейдите во вкладку CSS;
  • Внесите исправления вручную и нажмите сохранить;
  • После внесения исправлений вручную использование
    визуального редактора CSS - запрещено.
